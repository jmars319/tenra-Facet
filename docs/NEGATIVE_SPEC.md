# Negative Spec

Facet by Tenra is not an answer engine. The following constraints are product constraints, not optional style choices.

- Never summarize the web for the user as if one compressed answer is sufficient.
- Never answer instead of orienting.
- Never persuade the user toward one conclusion.
- Never collapse ambiguity before the user has had a chance to inspect it.
- Never hide which provider surfaced a result.
- Never blur the difference between a result and an orientation block.
- Never present reframing as fact-finding or verdict-making.
- Never remove the user’s ownership of interpretation.
- Never pretend a mock, provider, or generated artifact is first-party truth.
- Never convert safety or refusal behavior into silent deletion of user context.
